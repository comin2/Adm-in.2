Tabs

If you want to display some tabs, you may use the Admin2.Tabs plugin.

Automatic setup

{% highlight html %}
Tab 1 Tab 2 Tab 3

Tab 1

Tab 2

Tab 3

{% endhighlight %}

The HTML pretty much looks like it. The tabs-container allows the plugin to automatically start when the page is loaded.

By default the first tab of the container is shown.

Manual setup

If you need to manually create a tab functionnality, you can use this function:

{% highlight javascript %} var container = document.getElementById('#my-tabs-container'); Admin2.Tabs.setSingleTabs(container); {% endhighlight %}

The (first and only) parameter is the container. And that's it.

Show a specific tab

You may need to show a specific tab manually.

{% highlight javascript %} Admin2.Tabs.showTab('#my-tab'); {% endhighlight %}

The first (required) parameter is the selector to the tab. The function automatically finds the container the everything (as long as the HTML is properly formatted).

As a second (optional) parameter you can pass an event that would be stopped. Simple as that!
